Variables

Most of the time JavaScript needs to work with information. Examples:

A shoe store: The information would include shoes being sold and a shopping cart.

Social app:Users, messages, inbox, profile, etc.

Variables are used to store this information. Also called "Named storage."

Use "let" most of the time, but use "const" when declaring a constant (unchanging) variable.

let message;

message = 'Hello';

(The string 'Hello' is being stored in the variable named message.)

alert(message)

Multiple variables in one line:

let user = 'Name', age = Number, message = Birthday Message

OR

let user = 'Name';

let age = Number;

let message = Birthday Message;

OR

let user = 'Name',

age = Number,

message = Birthday Message;

Combine two variables into each other:

let hello = "Hello";

let message = " World!";

alert(hello + message)
