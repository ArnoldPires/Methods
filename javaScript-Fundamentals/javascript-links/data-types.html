Data types

A value in JavaScript is always of a certain type. Like a string or a number. There are 8 of them.

  • Represents both integer and floating numbers
  • A phrase surrounded by quotes `Hello, ${name}`
  • This type only has two values: true and false
  • A special value that represents nothing, empty, or unknown value
  • When a value is not assigned
  • Used to store a collection of data
  • For unique identifiers
  • Allows us to see which type is stored in a variable

Any one of these data types can be placed in a variable.
