Filter Method

What is the every method in javascript?

The filter() method creates a new array that contains a subset of elements of the original array.

Syntax of the filter() method: arrayObject.filter(callback, contextObject)

This method is usually used to remove or filter out specific items in an array, depending on your code

Click the button below to filter out the ages for people who can vote:

Ages of people who want to vote: 15, 24, 56, 17, 19, 44, 36

Answer

document.querySelector("#exampleOne").addEventListener("click", giveAnswerOne);
function canVote(age) {
return age >= 18;
}
function giveAnswerOne() {
let filtered = [15, 24, 56, 17, 19, 44, 36].filter(canVote);
document.querySelector("#answerOne").innerHTML = filtered;
}
